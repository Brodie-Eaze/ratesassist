## Working name

**RatesAssist** is the confirmed working name. This document records the considered alternatives, the rationale, and the availability checks required before locking the brand legally.

---

## Why "RatesAssist" works

| Property | Score |
|---|---|
| Says what it does | Strong — "rates" + "assist" is unambiguous |
| Australian flavour | Strong — "rates" is the AU/NZ term, not US "property tax" |
| Memorability | Solid — short, two-word compound |
| Pronunciation | Clean — no ambiguity |
| Domain availability prospect | Likely partial — `.com` may be taken, `.com.au` likely available |
| Trademark prospect | Probable — but check |
| Brand extensibility | Strong — RatesRecovery, RatesIntel, RatesChat are natural product line extensions |
| Council buyer perception | Professional, focused, not gimmicky |

The name signals exactly what we do, in the language the buyer uses. This is a public-sector market — clarity beats cleverness.

---

## Considered alternatives (and why we passed)

| Name | Why considered | Why passed |
|---|---|---|
| **CouncilCo** | Broad, extensible | Too generic; doesn't say what we do; thousands of "councilco" -style entities |
| **RatesPilot** | Implies guided assistance | Slightly heroic / startup-y; less neutral for govt buyers |
| **AssessIQ** | Plays on "assessment" + intelligence | Cute but oblique; rates officers don't say "assess" — they say "rate" |
| **TenementSense** | Captures the recovery thesis | Too narrow — only describes one feature; alienates non-mining councils |
| **RatesGenie** | Wish-fulfilment metaphor | Frivolous for govt buyers; "genie" trademarks are crowded |
| **Civica*Anything*** | Plays on existing brand recognition | Trademark risk — Civica is a competing-platform vendor |
| **TechOne *Anything*** | Same | Same trademark risk |
| **OpenRates** | Open-source vibe | Implies open data / open source; we're proprietary |
| **Strata** | Different vertical option | Already a major industry term; trademark crowded |
| **Levy** | Short, evocative | Tax / finance overlap; too overloaded |
| **Rateline** | Customer-service feel | Sounds like a hotline |
| **RateBoard** | Dashboard-y | Limits to one product surface |
| **Aurean Rates** | Sub-brand of existing entity | Buries the product behind another brand; weakens RatesAssist as a category brand |

## Availability checks — to run before locking

Run all checks. A green tick on every item is required before treating the name as locked.

### 1. ASIC company name

- [ ] Search https://connectonline.asic.gov.au/RegistrySearch/faces/landing/SearchRegisters.jspx
- [ ] Check **RatesAssist Pty Ltd** is available
- [ ] Check **Rates Assist Pty Ltd** (with space) is also available
- [ ] If both available, reserve immediately ($55 ASIC fee)

### 2. Domain availability

- [ ] `ratesassist.com` — check via auDA-accredited registrar or namecheap
- [ ] `ratesassist.com.au` — must be ABN-tied; available after ABN exists
- [ ] `rates.assist` — check `.assist` TLD pricing (~$50–80/yr typical)
- [ ] `getratesassist.com` and `tryratesassist.com` — backup options
- [ ] `ratesassist.app` and `ratesassist.io` — secondary technical domains

Priority: secure `.com` and `.com.au` at minimum.

### 3. IP Australia trademark search

- [ ] Search https://search.ipaustralia.gov.au for **RatesAssist** (Class 9 — software, Class 42 — SaaS, Class 35 — business consultancy)
- [ ] Search variations: "Rates Assist", "RATES ASSIST"
- [ ] Search phonetically similar marks
- [ ] If clear, file application — ~$330 per class via online application
- [ ] Consider classes 9, 35, 42 at minimum (cover software, business services, SaaS)

### 4. Phonetic / cultural check

- [ ] Pronunciation in AU English — clean
- [ ] Pronunciation in NZ English — clean (we'll expand there)
- [ ] No unfortunate homophones in major Indigenous Australian languages where we'll deploy (especially WA — speak with mum's contacts)
- [ ] No cultural sensitivity issues identified

### 5. Existing-use scan

- [ ] Search LinkedIn — any existing companies using the name
- [ ] Search GitHub for `ratesassist` — note any open-source projects (low risk but flag)

## Brand identity — initial direction

Defer detailed brand work until pilot is signed. For the pre-pilot period, working visual standards:

- **Wordmark:** "RatesAssist" set in a clean, geometric sans-serif (Inter, Söhne, or similar)
- **Colour palette:** Prefer trust-signalling tones. Blue/grey base, with an accent that distinguishes from the typical AU government palette. Avoid red (collections / debt connotation) and bright greens (clichéd "recovery" framing).
- **Tone of voice:** Direct, precise, confident. No jargon. No emojis. Council buyers respect calm professionalism over startup energy.
- **Logo:** Defer. A clean wordmark is sufficient until brand identity work is commissioned post-pilot.

## Domain investment

Budget $400–800 in year 1 for defensive domain registrations:

- `ratesassist.com` — primary
- `ratesassist.com.au` — primary (AU)
- `ratesassist.co` and `ratesassist.app` — defensive
- Common typos (`rateassist.com`, `ratesasist.com`) — cheap defensive
- Country variants if expanding (`ratesassist.co.nz` for NZ year 2+)
